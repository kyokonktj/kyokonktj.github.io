オンライン古書店のランディングページデザイン

制作期間
約2週間
ツール
AdobeXD, Visual Studio Code
point
Amazonマーケットプレイス上に店舗を構えて運営しているが、業務内容やコンセプトを伝えるページがないので、作ってほしいという想定。時代小説の取り扱いが多いので、リタイアして時間に余裕のある年配層をターゲットにしました。
point
運営元が就労移行支援事業所であるため、一つ一つの本を真心をこめて届ける真面目さを前面に出し、信頼感を与えるように工夫しました。手描きのイラストを配置して手作り感を出し、明朝体のフォントで知的な雰囲気を出しました。
